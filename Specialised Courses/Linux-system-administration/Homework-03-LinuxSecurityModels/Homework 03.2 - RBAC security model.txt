2. Опишете какво представлява RBAC security модела

RBAC (Role-Basec Access Control) - Security модел при който достъпа до системата се позволява на оторизирани потребители.
Използва се при повече компании с повече от 500 души персонал. В една организация се създават роли за различни длъжности.
Позволението за изпълнение на дадени операции се предоставя на специфични роли. Тъй като на потребителите не се възлагат
директно разрешения, а те ги получават според ролята или ролите си, управлението на индивидуалните потребителски права става
лесно.
